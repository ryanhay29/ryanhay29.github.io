Overview

I have contributed to several projects at the Human Augmentation via Dexterity Engineering Research Center (HAND ERC), as an undergraduate research assistant.

I built a database of commercial robotic hands including specifications and images hosted on GitHub Pages. The database provides standardized information across many hands to support comparison and benchmarking.

I also developed a Qt interface in C++ to control a 3D printed Robot Nano Hand using sliders and configurable presets, shown in the demo video in Fig. 1. The interface allows researchers to control the hand in real time to assess its dexterity and performance.

Currently, I am collecting tactile data using Daimon Robotic claw grippers performing complex tasks, which will be used to train a robot with tactile sensors to carry out tasks autonomously.
